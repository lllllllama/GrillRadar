### 问题 1
**角色**: 导师/PI
**标签**: 研究动机与创新性

**问题**: 你在HMS-Former中提出了"Adaptive Scale Fusion"和"Cross-Scale Attention"模块。请详细说明:1)这两个模块的设计动机是什么? 解决了现有方法的什么问题? 2)与FPN、BiFPN等经典多尺度融合方法的本质区别是什么? 3)你如何证明这是真正的创新,而不是现有模块的简单组合?

**提问理由**: 这是PhD面试的核心问题,考察研究的原创性和深度。导师想了解:1)你是否真正理解问题本质; 2)是否做了充分的文献调研,知道自己的工作与现有方法的区别; 3)是否有科学的验证方法(如消融实验)证明创新点的有效性。这是区分"做实验的学生"和"有科研潜力的研究者"的关键。

**基准答案**:
一个好的回答应包含:
1. **设计动机**(问题驱动):
   - 现有问题: FPN等方法使用简单的加法/拼接融合,未考虑不同尺度特征的重要性差异
   - 观察发现: 在医疗影像中,不同器官的最优分割尺度不同(如肝脏需要全局信息,血管需要细节)
   - 动机: 设计自适应的融合机制,让网络根据输入动态调整尺度权重
2. **与现有方法的区别**:
   - FPN: 固定的top-down融合,无learnable权重
   - BiFPN: 引入learnable权重,但是静态的(不随输入变化)
   - 我们的Adaptive Fusion: 使用轻量级网络(如attention模块)根据特征内容动态预测融合权重
3. **创新性验证**:
   - 消融实验: 对比FPN、BiFPN、Ours,证明性能提升
   - 可视化: 展示不同样本的尺度权重分布,证明自适应性
   - 理论分析: 从信息论或优化角度解释为什么有效
4. **诚实面对局限**: 如承认"创新点相对incremental,但在医疗场景下很有效"

**支撑材料**:
- 关键论文: FPN, PANet, BiFPN, NAS-FPN, Dynamic Convolution
- 推荐阅读: 《多尺度特征融合综述》, CVPR/ICCV分割相关论文

### 问题 2
**角色**: 导师/PI
**标签**: 实验设计与科学严谨性

**问题**: 你提到HMS-Former在Synapse数据集上达到SOTA (Dice 82.7%)。请详细说明:1)你的实验设置是什么(训练集/测试集划分、数据增强、超参数)? 2)如何保证实验的公平性和可重复性? 3)除了Dice,你还报告了哪些指标? 为什么? 4)有没有做统计显著性检验?

**基准答案**:
一个好的回答应包含:
1. **实验设置**:
   - 数据集: Synapse多器官分割,30个CT扫描,18个训练,12个测试(标准划分)

### 问题 5
**角色**: 导师/PI
**标签**: 数学与理论基础

**问题**: Dice Loss是医疗分割中常用的损失函数。请从数学角度解释:1)Dice Loss的公式和梯度是什么? 2)为什么Dice Loss对类别不平衡问题有帮助? 3)Dice Loss有什么局限性? 如何改进?

**提问理由**: PhD不能只会调包,需要有扎实的数学基础。导师想考察:1)是否理解常用方法的数学原理; 2)是否有推导和分析能力; 3)是否能从理论角度思考改进方向。这是区分"工程师"和"研究者"的标志。

**基准答案**:
一个好的回答应包含:
1. **数学公式**:
   - Dice Coefficient: DSC = 2|X∩Y| / (|X|+|Y|)
   - Dice Loss: L = 1 - DSC = 1 - (2ΣpᵢGᵢ + ε) / (Σpᵢ² + ΣGᵢ² + ε)
   - 梯度: ∂L/∂pᵢ = -2(Gᵢ(Σpⱼ²+ΣGⱼ²) - pᵢΣGⱼ) / (Σpⱼ²+ΣGⱼ²)²
2. **类别不平衡帮助**:
   - Dice是基于intersection和union,对正负样本比例不敏感
   - 与Cross Entropy相比,CE对每个像素单独计算,受样本量影响大
   - Dice关注overlap,即使正样本少,只要overlap好,loss也小
3. **局限性**:
   - 对小目标敏感: 小目标的少量像素错误会导致Dice大幅下降
   - 梯度不稳定: 分母接近0时,梯度可能爆炸或消失
   - 无法区分边界质量: 只关注overlap,不关注边界精度
4. **改进方向**:
   - 加入边界损失(Boundary Loss, Hausdorff Distance Loss)
   - Focal Dice Loss: 对难样本加权
   - Tversky Loss: 可调节FP和FN的权重
   - 组合损失: Dice + CE, Dice + Boundary

**支撑材料**:
- 关键论文: V-Net (首次提出Dice Loss), Boundary Loss, Hausdorff Distance Loss
- 推荐阅读: 《深度学习中的损失函数设计》, 分割损失函数综述
- 搜索关键词: "dice loss gradient", "loss functions for segmentation"

**练习提示词**:

### 问题 11
**角色**: 技术面试官
**标签**: 深度学习基础

**问题**: 请解释Transformer中的Self-Attention机制的数学原理,包括Q、K、V的计算,以及Multi-Head Attention的作用。在图像分割中,Transformer相比CNN有什么优势和劣势?

**提问理由**: Transformer是CV领域的核心技术,PhD学生需要深入理解。导师想考察:1)是否理解数学原理,而非只会调包; 2)是否能分析不同架构的优劣; 3)是否能根据任务特点选择合适的模型。

**基准答案**:
一个好的回答应包含:
1. **Self-Attention数学**:
   - Q = XWq, K = XWk, V = XWv (线性变换)
   - Attention(Q,K,V) = softmax(QK^T/√dk)V
   - 含义: 每个token根据与其他token的相似度(QK^T)加权聚合信息(V)
   - √dk缩放: 防止内积过大导致softmax梯度消失
2. **Multi-Head Attention**:
   - 并行计算h个attention,每个关注不同的子空间
   - 增强表达能力,捕获多种关系(如全局结构、局部细节)
   - Concat多个head,再线性变换
3. **Transformer vs CNN**:
   - 优势:
     - 全局感受野: 一层就能建模全局依赖,CNN需要堆叠多层
     - 灵活性: 注意力权重动态计算,适应不同输入
     - 并行性: 相比RNN,Transformer可完全并行
   - 劣势:
     - 计算复杂度: O(n²),对高分辨率图像(如医疗影像)计算量大
     - 数据需求: 需要大量数据预训练,小数据集容易过拟合
     - 归纳偏置弱: 缺少CNN的局部性和平移不变性,需要更多数据学习
4. **分割应用**: 通常使用Hybrid架构(如TransUNet),CNN提取局部特征,Transformer建模全局依赖

**支撑材料**:
- 关键论文: Attention is All You Need, ViT, Swin Transformer
- 推荐阅读: 《Transformer详解》, Jay Alammar的博客
- 搜索关键词: "transformer explained", "self-attention mechanism"
