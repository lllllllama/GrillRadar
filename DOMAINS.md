# GrillRadar 支持的领域列表

> **Milestone 3**: 配置驱动的领域管理（TrendRadar式）

## 如何使用

在生成报告时，可以通过以下方式指定领域：

### CLI方式

```json
{
  "mode": "job",
  "target_desc": "字节跳动后端开发工程师",
  "domain": "backend",
  "level": "junior"
}
```

### Web界面

在"技术领域"下拉框中选择对应的领域即可。

---

## 领域配置的作用

指定领域后，GrillRadar会：

1. **注入领域知识**: 在Prompt中加入该领域的关键词、技术栈、经典论文等
2. **聚焦问题范围**: 生成的问题会更聚焦于该领域的核心能力
3. **提供专业参考**: support_notes中会包含该领域的推荐阅读和学习资源
4. **匹配典型岗位**: 根据领域的典型岗位要求调整问题难度和方向
